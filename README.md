Scenario chosen for the first homework from web design: Local small business digital presence.

This project is a 4 page website for a fictional bakery: **Artigio Bakery**. It was created using only HTML5 and CSS3 and includes a homepage, an about page, a data page with a real table, and a contact page with a labeled form.

Requirements implemented:
- 4 pages: `index.html`, `about.html`, `data.html`, `contact.html`
- semantic structure on every page: `header`, `nav`, `main`, `footer`
- one external stylesheet: `style.css`
- real HTML table with caption and header row
- contact form with visible labels
- consistent navigation and styling across pages
- accessibility elements such as skip link and alt text

Known limitations:
This website is static. The contact form does not send real data, and the product details like Instagram page and telephone number are fictional.
